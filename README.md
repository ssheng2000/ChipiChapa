# Chipi

https://github.com/user-attachments/assets/6b55fa59-b518-4845-86f8-c736330f0e40


A strategic puzzle-platformer built for the **"Set in Stone"** Game Jam. 

Chipi is still a child and doesn't have a sense of direction. Your goal is to manipulate the environment to help Chipi navigate the icy mountains and reach Chapa safely.

## 🎮 How to Play

The game alternates between **Action** and **Build Mode**.

1. **Build Mode (Spacebar):** Pause the game to enter Build Mode.
2. **Place Items:** Click and drag objects from your inventory onto the environment.
3. **Run:** Unpause the game and watch Chipi go!
4. **Stuck?** Press **R** to restart the level.

### **Controls**
| Key | Action |
| :--- | :--- |
| `Spacebar` | Toggle Build Mode (Pause / Unpause) |
| `Click + Drag` | Place/Move Items |
| `R` | Restart Level |

---

## 🛠️ The Toolkit

Use these items to guide Chipi through the environment:

* **🪨 Stone Block:** Flips Chipi's direction upon collision
* **🍄 Mushroom:** Launches Chipi into the air after pressing down
* **🦆 Duck:** Blows Chipi
